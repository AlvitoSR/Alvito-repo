# EmbedTV — Plugin CloudStream

Plugin de TV ao vivo para o [CloudStream 3](https://github.com/recloudstream/cloudstream).

## Como instalar o plugin

1. Vá em **Releases** nesse repositório
2. Baixe o arquivo `EmbedTvProvider.cs3`
3. No CloudStream: **Configurações → Extensões → ícone "+"** → selecione o arquivo

## Canais disponíveis

SporTV, ESPN, CazeTV, Globo, Record, SBT, Band, CNN Brasil, GloboNews e muito mais.
